# 第三方SDK信息共享清单

更新日期：2026年3月30日

本APP使用的第三方SDK及其信息收集情况如下：

## 1. 高德地图SDK
- 所属公司：高德软件有限公司
- 使用目的：天气自动定位、导航跳转
- 收集信息：设备位置信息、设备标识符、网络信息
- 隐私政策：https://lbs.amap.com/pages/privacy/

## 2. 百度地图SDK
- 所属公司：百度在线网络技术有限公司
- 使用目的：天气自动定位、导航跳转
- 收集信息：设备位置信息、设备标识符、网络信息
- 隐私政策：https://map.baidu.com/zt/client/privacy/

## 3. 微信SDK
- 所属公司：深圳市腾讯计算机系统有限公司
- 使用目的：分享功能
- 收集信息：设备标识符、网络信息
- 隐私政策：https://weixin.qq.com/cgi-bin/readtemplate?t=weixin_agreement&s=privacy

## 4. 极光推送
- 所属公司：深圳市和讯华谷科技有限公司
- 使用目的：消息推送
- 收集信息：设备标识符、网络信息
- 隐私政策：https://www.jiguang.cn/license/privacy

## 5. Flutter框架
- 所属公司：Google LLC
- 使用目的：APP开发框架
- 收集信息：无
- 隐私政策：https://policies.google.com/privacy

说明：
- 以上SDK仅在用户使用相关功能时才会激活
- 不使用相关功能时不会收集任何信息
- 所有SDK的数据收集遵循各自的隐私政策
