## 一、已声明权限（AndroidManifest.xml）

| 权限 | 索要理由（给用户看） | 详细说明 | 功能对应 |
|------|---------------------|---------|---------|
| `INTERNET` | 天气查询、AI对话 | 天气API、AI接口调用 | 天气、AI待办解析 |
| `RECORD_AUDIO` | 语音输入待办 | 仅点击麦克风时使用，本地识别不上传 | 待办语音输入 |
| `ACCESS_FINE_LOCATION` | 天气自动定位、导航跳转、到达提醒 | 到达指定地点推送相关待办 | 天气、导航、到达提醒 |
| `ACCESS_COARSE_LOCATION` | 同上（粗略定位） | 同上 | 同上 |
| `POST_NOTIFICATIONS` | 待办提醒、用药提醒、每日汇总 | Android 13+ 必须声明 | 提醒系统 |
| `SCHEDULE_EXACT_ALARM` | 精确定时提醒 | Android 12+ 精确闹钟 | 待办定时提醒 |
| `USE_EXACT_ALARM` | 精确定时提醒 | 同上 | 同上 |
| `RECEIVE_BOOT_COMPLETED` | 重启后恢复提醒任务 | 开机后自动恢复定时提醒 | 提醒系统 |
| `WAKE_LOCK` | 后台提醒唤醒设备 | 定时提醒时唤醒屏幕 | 提醒系统 |
| `ACTIVITY_RECOGNITION` | 读取步数同步到运动模块 | 读取手机计步器数据 | 运动tab |
| `READ_EXTERNAL_STORAGE` | 导入微信/支付宝账单CSV | 用户主动选择文件，本地解析 | 账单导入 |
| `WRITE_EXTERNAL_STORAGE` | 导出记账数据Excel | 生成文件保存到手机 | 记账导出 |
| `CAMERA` | 扫码比价、拍照识别发票 | 照片仅本地处理，不上传 | 扫码比价、OCR |

## 二、权限使用时机

### 首次启动引导
- 弹出底部弹窗，展示所有权限说明
- 隐私承诺醒目展示
- 用户可选择"去授权"或"跳过"

### 功能触发时请求
| 权限 | 触发时机 | 弹窗文案 |
|------|---------|---------|
| 麦克风 | 点击语音按钮 | "用于语音输入待办，动动嘴就能添加任务" |
| 位置 | 首次查看天气 | "用于天气自动定位、导航跳转" |
| 通知 | 首次设置提醒 | "用于待办提醒、用药提醒、每日汇总推送" |
| 步数 | 首次打开运动tab | "用于读取步数，同步到运动习惯模块" |
| 存储 | 首次导入账单 | "用于导入微信/支付宝账单CSV" |
| 相机 | 首次使用扫码 | "用于扫码比价、拍照识别发票" |

### 设置页权限管理
- 每个权限可查看详细说明
- 点击跳转系统设置关闭/开启

## 五、应用商店审核注意事项

### Google Play
- 需要在 Play Console 填写每个权限的使用说明
- `CAMERA` 权限需要说明具体用途
- `READ_EXTERNAL_STORAGE` 在 Android 11+ 需要用 `MANAGE_EXTERNAL_STORAGE` 或 SAF

### 国内应用商店
- 华为/小米/OPPO/vivo 都需要隐私政策链接
- 首次启动必须展示隐私协议弹窗（用户同意后才能收集任何信息）
- 需要提交《个人信息收集清单》和《第三方SDK信息共享清单》

### 隐私政策要点
```
1. 数据存储：全部本地存储，不上传服务器
2. 权限使用：逐项说明每个权限的用途
3. 第三方SDK：列出使用的SDK及其隐私政策
4. 用户权利：用户可随时关闭权限、删除数据
5. 联系方式：提供开发者联系方式
```
