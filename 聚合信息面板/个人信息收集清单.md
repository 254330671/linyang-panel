# 个人信息收集清单

更新日期：2026年3月30日

根据《个人信息保护法》及相关规定，现将本APP收集使用个人信息的目的、方式和范围公示如下：

## 一、基本信息

| 信息类型 | 收集目的 | 是否必须 | 存储方式 |
|---------|---------|---------|---------|
| 年龄 | 个性化健康建议/里程碑推荐 | 否 | 本地存储 |
| 性别 | 经期追踪/个性化内容 | 否 | 本地存储 |
| 职业 | 职业规划/预算模板推荐 | 否 | 本地存储 |
| 身高体重 | BMI计算/健康分析 | 否 | 本地存储 |
| 血型 | 急救卡展示 | 否 | 本地存储 |
| 过敏史 | 急救卡展示/健康预警 | 否 | 本地存储 |
| 慢性病 | 急救卡展示/健康分析 | 否 | 本地存储 |

## 二、健康信息

| 信息类型 | 收集目的 | 是否必须 | 存储方式 |
|---------|---------|---------|---------|
| 睡眠记录 | 睡眠质量分析 | 否 | 本地存储 |
| 饮水记录 | 饮水量追踪 | 否 | 本地存储 |
| 症状记录 | 健康趋势分析 | 否 | 本地存储 |
| 用药记录 | 用药提醒/频率预警 | 否 | 本地存储 |
| 经期记录 | 周期预测/症状关联 | 否 | 本地存储 |
| 运动记录 | 运动习惯追踪 | 否 | 本地存储 |
| 就医记录 | 就医信息管理 | 否 | 本地存储 |
| 体检记录 | 健康指标追踪 | 否 | 本地存储 |
| 心情记录 | 情绪趋势分析 | 否 | 本地存储 |

## 三、生活信息

| 信息类型 | 收集目的 | 是否必须 | 存储方式 |
|---------|---------|---------|---------|
| 待办事项 | 任务管理 | 是 | 本地存储 |
| 记账记录 | 收支管理 | 否 | 本地存储 |
| 习惯记录 | 习惯追踪 | 否 | 本地存储 |
| 写给未来的信 | 个人记录 | 否 | 本地存储 |

## 四、设备权限

| 权限 | 使用目的 | 是否必须 |
|------|---------|---------|
| 麦克风 | 语音输入待办 | 否 |
| 位置 | 天气定位、导航、到达提醒 | 否 |
| 通知 | 待办提醒、用药提醒、每日汇总 | 否 |
| 身体活动 | 读取步数同步到运动模块 | 否 |
| 存储 | 导入账单CSV、导出记账数据 | 否 |
| 相机 | 扫码比价、拍照识别发票 | 否 |

## 五、数据存储说明

- 所有数据仅存储在用户设备本地
- 不上传到任何服务器
- 开发者无法访问用户数据
- 用户可随时删除所有数据
