## 三、推荐立即加入的功能

### 1. 汇率换算
```javascript
// 免费API
const API = 'https://api.exchangerate-api.com/v4/latest/USD';
// 返回所有货币对美元的汇率
```

### 2. 二维码生成
```javascript
// 使用 qrcode.js
QRCode.toCanvas(canvas, 'https://example.com');
```

### 3. 农历显示
```javascript
// 使用 lunar-javascript
const lunar = Lunar.fromDate(new Date());
console.log(lunar.getYearInChinese()); // 丙午年
console.log(lunar.getMonthInChinese()); // 二月
console.log(lunar.getDayInChinese()); // 初八
```

### 4. Excel导出
```javascript
// 使用 SheetJS
const ws = XLSX.utils.json_to_sheet(data);
const wb = XLSX.utils.book_new();
XLSX.utils.book_append_sheet(wb, ws, "记账");
XLSX.writeFile(wb, "记账.xlsx");
```

### 5. 语音播报
```javascript
// 浏览器原生
const utterance = new SpeechSynthesisUtterance('你有一个待办需要处理');
speechSynthesis.speak(utterance);
```
