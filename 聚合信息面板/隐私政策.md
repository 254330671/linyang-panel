## 一、我们收集的信息

本APP所有数据仅存储在你的设备本地，不会上传到任何服务器。

### 1.1 设备权限

| 权限 | 用途 | 是否必须 |
|------|------|---------|
| 麦克风 | 语音输入待办 | 否 |
| 位置 | 天气定位、导航、到达提醒 | 否 |
| 通知 | 待办提醒、用药提醒、每日汇总 | 否 |
| 身体活动 | 读取步数同步到运动模块 | 否 |
| 存储 | 导入账单CSV、导出记账数据 | 否 |
| 相机 | 扫码比价、拍照识别发票 | 否 |

### 1.2 本地存储数据

- 待办事项
- 记账记录
- 健康档案（身高/体重/血型/过敏史/慢性病）
- 症状/用药/经期/睡眠/运动/就医/体检记录
- 心情/习惯记录
- 写给未来的信
- 个人档案（年龄/性别/职业）

### 1.3 第三方服务

用户自行配置AI接口（DeepSeek/OpenAI等）时，输入文本会发送到对应第三方服务，我们不会存储这些请求内容。

## 二、如何使用信息

所有数据仅用于在你的设备上提供APP功能，不用于任何其他目的。

## 三、如何存储信息

- 加密保护：敏感数据支持用户设置密码加密
- 我们无法访问你的任何数据
