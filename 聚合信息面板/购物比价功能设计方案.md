# 购物比价功能设计方案

> 独立文档，供后续提取使用
> 创建日期：2026-03-30

---

## 一、核心问题

同一商品在淘宝/京东/拼多多没有统一ID，无法直接跨平台比价。

---

## 二、可行方案

### 方案E（推荐）：比价动作

**思路**：用户输入商品关键词 → 系统同时生成淘宝/京东/拼多多搜索链接 → 用户自己打开对比

**技术实现**：
```javascript
// 用户输入商品名称/关键词
const keyword = "iPhone 16 Pro Max 256G";

// 生成三个平台的搜索链接
const links = {
  taobao: `taobao://s.taobao.com?q=${encodeURIComponent(keyword)}`,
  jd: `openapp.jdmobile://virtual?params={"category":"jump","des":"search","keyword":"${encodeURIComponent(keyword)}"}`,
  pdd: `pinduoduo://search?keyword=${encodeURIComponent(keyword)}`,
  // H5降级
  taobao_h5: `https://s.taobao.com/search?q=${encodeURIComponent(keyword)}`,
  jd_h5: `https://so.m.jd.com/ware/search.action?keyword=${encodeURIComponent(keyword)}`,
  pdd_h5: `https://mobile.yangkeduo.com/search_result.html?search_key=${encodeURIComponent(keyword)}`
};
```

**用户体验**：
1. 用户点击"比价"按钮
2. 输入商品名称（或从待办自动提取）
3. 系统生成三个平台的搜索链接
4. 用户点击分别打开 → 手动对比价格

---

### 其他方案（暂不推荐）

| 方案 | 难度 | 原因 |
|------|------|------|
| 自己爬取 | ❌ 极高 | 法律风险+反爬机制 |
| 官方API | ❌ 高 | 需企业资质，MVP阶段不现实 |
| 第三方API | ⚠️ 中 | 需商务合作（慢慢买/什么值得买） |

## 三、扩展功能

### 1. 历史价格曲线
- 接入慢慢买数据
- 告诉用户"当前价格是否历史最低"

### 2. 优惠券聚合
- 接入淘宝联盟/京东联盟
- 帮用户找优惠券

### 3. 降价提醒
- 用户标记商品
- 定期检查是否有降价通知

## 四、代码示例

### 前端调用
```javascript
// 一键比价
function comparePrice(keyword) {
  const links = generateSearchLinks(keyword);
  // 展示弹窗，用户选择平台
  showCompareModal(links);
}

// 生成搜索链接
function generateSearchLinks(keyword) {
  return {
    taobao: {
      app: `taobao://s.taobao.com?q=${encodeURIComponent(keyword)}`,
      h5: `https://s.taobao.com/search?q=${encodeURIComponent(keyword)}`,
      name: '淘宝'
    },
    jd: {
      app: `openapp.jdmobile://virtual?params={"category":"jump","des":"search","keyword":"${encodeURIComponent(keyword)}"}`,
      h5: `https://so.m.jd.com/ware/search.action?keyword=${encodeURIComponent(keyword)}`,
      name: '京东'
    },
    pdd: {
      app: `pinduoduo://search?keyword=${encodeURIComponent(keyword)}`,
      h5: `https://mobile.yangkeduo.com/search_result.html?search_key=${encodeURIComponent(keyword)}`,
      name: '拼多多'
    }
  };
}
```

### 待办联动
```javascript
// 从待办中提取购物关键词
function extractShoppingKeyword(todo) {
  // 例如：待办"买牛奶" → 关键词"牛奶"
  const patterns = [
    /买(.+)/,
    /购买(.+)/,
    /下单(.+)/,
    /看看(.+)/
  ];
  
  for (const pattern of patterns) {
    const match = todo.match(pattern);
    if (match) return match[1].trim();
  }
  
  return todo;
}
```
