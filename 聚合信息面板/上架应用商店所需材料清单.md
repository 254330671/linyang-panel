# 上架应用商店所需材料清单

整理日期：2026年3月30日

---

## 一、Google Play（安卓）

### 1.1 开发者账号
- [ ] 注册Google Play开发者账号
- [ ] 费用：一次性25美元
- [ ] 需要：Google账号 + 信用卡
- [ ] 注册地址：https://play.google.com/console/signup

### 1.2 APP信息
- [ ] APP名称（中英文）
- [ ] APP描述（短描述80字 + 长描述4000字）
- [ ] APP图标（512x512 PNG）
- [ ] 特色图片（1024x500）
- [ ] 截图（手机+平板，至少2张，最多8张）
- [ ] 应用分类
- [ ] 内容分级问卷

### 1.3 合规材料
- [ ] 隐私政策链接（已生成）
- [ ] 数据安全声明（在Play Console填写）
- [ ] 目标受众群体声明
- [ ] 广告声明（无广告）

### 1.4 APK要求
- [ ] 签名的AAB格式（Android App Bundle）
- [ ] 版本号 >= 1
- [ ] targetSdkVersion >= 34（Android 14）
- [ ] 64位支持

### 1.5 审核时间
- 首次提交：通常3-7天
- 更新：通常几小时到1天

## 二、Apple App Store（iOS）

### 2.1 开发者账号
- [ ] 注册Apple Developer Program
- [ ] 费用：每年99美元（个人）或299美元（企业）
- [ ] 需要：Apple ID + 信用卡
- [ ] 注册地址：https://developer.apple.com/programs/
- [ ] 需要邓白氏编码（DUNS Number，企业需要，个人可选）

### 2.2 APP信息
- [ ] APP名称（30字符以内）
- [ ] 副标题（30字符以内）
- [ ] APP描述（4000字符以内）
- [ ] 关键词（100字符以内，逗号分隔）
- [ ] APP图标（1024x1024 PNG，无圆角）
- [ ] 截图（6.7寸/6.5寸/5.5寸，至少1组）
- [ ] 预览视频（可选，30秒以内）
- [ ] 应用分类
- [ ] 版权信息

### 2.3 合规材料
- [ ] 隐私政策链接（已生成）
- [ ] 隐私营养标签（在App Store Connect填写）
- [ ] App隐私详情
- [ ] 出口合规信息
- [ ] 内容版权确认

### 2.4 构建要求
- [ ] Xcode构建的IPA文件
- [ ] 需要Mac电脑 + Xcode
- [ ] 或使用CI/CD服务（如Codemagic、Bitrise）
- [ ] 需要Provisioning Profile和证书

### 2.5 审核时间
- 首次提交：通常1-7天
- 可能被拒需要修改重新提交

### 2.6 常见被拒原因
- 崩溃或bug
- 缺少隐私政策
- 功能不完整
- 使用私有API
- 缺少测试账号

## 三、国内安卓商店

### 3.1 华为应用市场
- [ ] 注册华为开发者账号（免费）
- [ ] 需要：手机号 + 实名认证
- [ ] 需要：软件著作权（可后补）
- [ ] 需要：隐私政策 + 用户协议（已生成）
- [ ] 需要：APP截图 + 图标
- [ ] 需要：APK签名文件

### 3.2 小米应用商店
- [ ] 注册小米开发者账号（免费）
- [ ] 需要：手机号 + 实名认证
- [ ] 需要：隐私政策 + 用户协议
- [ ] 需要：APP截图 + 图标

### 3.3 OPPO/vivo应用商店
- [ ] 注册对应开发者账号（免费）
- [ ] 需要：手机号 + 实名认证
- [ ] 需要：隐私政策 + 用户协议
- [ ] 需要：APP截图 + 图标

### 3.4 国内商店共同要求
- [ ] 隐私政策链接
- [ ] 用户协议
- [ ] 个人信息收集清单（已生成）
- [ ] 第三方SDK清单（已生成）
- [ ] APP截图（至少4张）
- [ ] APP图标
- [ ] 软件著作权（可后补，但建议尽快办理）

### 3.5 审核时间
- 华为：1-3天
- 小米：1-3天
- OPPO/vivo：2-5天
