## 加密模块集成指南

### 文件说明

- `encryption.js` — AES-256-GCM 加密模块（纯 JS，零依赖）
- `encryption-example.html` — 测试页面（浏览器打开即可试用）

### 集成到网页版的步骤

#### 1. 引入脚本

在 `index.html` 的 `<script>` 标签前加入：
```html
<script src="encryption.js"></script>
```

#### 2. 修改数据加载流程

原来：
```js
const raw = localStorage.getItem('mood');
if (raw) moodData = JSON.parse(raw);
```

改为：
```js
const raw = localStorage.getItem('mood');
if (raw) {
  const decrypted = await Encryption.unwrapFromStorage('mood', raw);
  moodData = JSON.parse(decrypted);
}
```

#### 3. 修改数据保存流程

原来：
```js
localStorage.setItem('mood', JSON.stringify(moodData));
```

改为：
```js
const encrypted = await Encryption.wrapForStorage('mood', JSON.stringify(moodData));
localStorage.setItem('mood', encrypted);
```

#### 4. 添加密码设置 UI

在设置页面添加：
```html
<div class="card">
  <h3>🔐 数据加密</h3>
  <input type="password" id="encrypt-pwd" placeholder="设置加密密码">
  <button onclick="setupEncryption()">开启加密</button>
  <button onclick="disableEncryption()">关闭加密</button>
  <p style="font-size:12px;color:var(--text2)">
    加密后敏感数据（心情、症状、经期、睡眠等）将被保护。<br>
    ⚠️ 忘记密码数据将无法恢复！
  </p>
</div>
```

```js
async function setupEncryption() {
  const pwd = document.getElementById('encrypt-pwd').value;
  if (!pwd || pwd.length < 6) {
    alert('密码至少 6 位');
    return;
  }
  
  await Encryption.setPassword(pwd);
  
  // 迁移已有数据
  const allKeys = Object.keys(localStorage);
  for (const key of allKeys) {
    if (Encryption.needsEncryption(key)) {
      const raw = localStorage.getItem(key);
      if (raw && !raw.startsWith('ENC:')) {
        const encrypted = await Encryption.wrapForStorage(key, raw);
        localStorage.setItem(key, encrypted);
      }
    }
  }
  
  // 保存验证令牌
  const token = await Encryption.generateVerificationToken();
  localStorage.setItem('enc_token', token);
  
  alert('✅ 加密已开启');
}

function disableEncryption() {
  Encryption.clearPassword();
  alert('已关闭加密（数据未解密，需手动处理）');
}
```

#### 5. 页面加载时请求密码

```js
// 页面初始化时检查是否需要密码
const token = localStorage.getItem('enc_token');
if (token) {
  // 弹出密码输入框
  const pwd = prompt('请输入加密密码');
  if (pwd) {
    const valid = await Encryption.verifyWithToken(token, pwd);
    if (valid) {
      await Encryption.setPassword(pwd);
      // 重新加载所有数据（这次会自动解密）
      loadAllData();
    } else {
      alert('密码错误');
    }
  }
}
```

### 需要加密的 key

```
mood, moodHistory, symptoms, meds, letters,
growthAlbum, sleepLog, periodLog, healthProfile,
bodyRecords, socialRecords
```

### 不加密的 key

```
weather, todos, habits, expenses, topics,
career, exams, skills, settings
```

### 密文格式

```
ENC:v2:base64(salt):base64(iv):base64(ciphertext+tag)
```

- `ENC:v2:` — 版本前缀
- `salt` — 16 字节随机盐（PBKDF2 用）
- `iv` — 12 字节随机 IV（GCM 用）
- `ciphertext+tag` — AES-256-GCM 密文 + 认证标签
