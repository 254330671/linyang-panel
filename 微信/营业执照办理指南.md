# 营业执照办理指南

> 适用场景：做 App（聚合信息面板），需要接微信/支付宝支付、上架应用商店
> 整理：2026-03-28

---

## 一、推荐类型：个体工商户

| 对比 | 个体工商户 | 有限责任公司 |
|------|-----------|-------------|
| **办理难度** | ⭐ 简单 | ⭐⭐⭐ 复杂 |
| **注册资金** | 无要求 | 认缴制（不用实缴） |
| **税务** | 简单，核定征收 | 需记账报税 |
| **责任** | 无限责任 | 有限责任 |
| **适合** | 个人/小团队起步 | 规模化、融资 |
| **微信支付** | ✅ 支持 | ✅ 支持 |
| **支付宝** | ✅ 支持 | ✅ 支持 |
| **应用商店上架** | ✅ 支持 | ✅ 支持 |

**结论：现阶段办个体工商户就够了。** 手续简单、成本低、维护省心。等做大了再升级公司。

## 二、办理所需材料

### 必备材料

| 材料 | 说明 |
|------|------|
| **身份证** | 经营者本人身份证原件+复印件 |
| **经营场所证明** | 见下方详细说明 |
| **申请表** | 当地市场监管局领取或网上下载 |
| **1寸照片** | 部分地区需要 |

### 经营场所（最关键的）

| 方式 | 说明 |
|------|------|
| **自有房产** | 房产证复印件 |
| **租赁** | 租赁合同 + 房东房产证复印件 |
| **网络经营** | 很多地区支持"仅网络经营"，不需要实体地址 |
| **住宅** | 部分城市允许住宅注册（需邻居/物业同意） |
| **集群注册** | 部分地区有创业园区提供地址挂靠（免费或几十元/年） |

> **注意：** 各地政策不同，建议先打电话问当地市场监管局（12315）确认需要什么。

## 三、经营范围怎么写

这是关键，写对了以后省很多事。建议包含：

```
软件开发；信息技术咨询服务；互联网信息服务；
计算机系统服务；网络技术服务；
电子产品销售；日用百货销售
```

### 每项对应什么用途

| 经营范围 | 对应场景 |
|---------|---------|
| 软件开发 | App 开发本身 |
| 信息技术咨询服务 | 提供技术服务 |
| 互联网信息服务 | **ICP 备案必须**，上架 App 必须 |
| 计算机系统服务 | 后端服务 |
| 网络技术服务 | 泛指技术类 |
| 电子产品销售 | 如果以后卖周边/硬件 |
| 日用百货销售 | 万一以后扩展业务 |

> **重点：一定要有"互联网信息服务"，否则办不了 ICP 备案和 App 上架。**

## 四、办理流程

### 方式一：线下办理

1. 去当地**市场监督管理局**（原来叫工商局）
2. 领表填写 + 提交材料
3. 审核（一般 1-3 个工作日）
4. 领取营业执照（纸质/电子）

### 方式二：网上办理（推荐）

1. 登录当地**政务服务网**（各省都有）
   - 全国通用：https://www.gsxt.gov.cn
   - 或搜索"XX省 一窗通 个体工商户"
2. 注册账号 → 在线填写信息
3. 上传身份证照片 + 经营场所证明
4. 在线签名（人脸识别）
5. 审核通过后下载**电子营业执照**
6. 纸质执照可邮寄或自取

### 办理时间

- 线下：1-3 个工作日
- 线上：1-5 个工作日
- **费用：免费**（工本费 0 元）

## 五、办完营业执照还要什么？

这是很多人忽略的，营业执照只是第一步。做 App 上架还需要：

### 1. ICP 备案（必须）

| 项目 | 说明 |
|------|------|
| **是什么** | 网站/App 在工信部的登记备案 |
| **为什么需要** | 2023 年起 App 必须有备案号才能上架 |
| **在哪办** | 你的服务器提供商（阿里云/腾讯云后台） |
| **你的情况** | 阿里云轻量服务器 → 阿里云后台直接申请 |
| **需要** | 营业执照 + 域名 + 身份证 |
| **时间** | 1-20 个工作日（看省份） |
| **费用** | 免费 |

**操作路径：** 阿里云控制台 → ICP 备案 → 提交资料 → 等审核 → 拿到备案号

### 2. 软件著作权（App 上架必须）

| 项目 | 说明 |
|------|------|
| **是什么** | 软件的"版权证"，证明这是你的作品 |
| **为什么需要** | 华为/小米/苹果等应用商店强制要求 |
| **在哪办** | 中国版权保护中心 |
| **网址** | https://www.ccopyright.com.cn |
| **类型** | 电子软著（快，1-2 周）或纸质软著（慢，1-3 个月） |
| **需要** | 营业执照 + 源代码 + 操作说明书 |
| **费用** | 电子免费，纸质 200-300 元 |

**建议办电子软著，快且免费。**

### 3. 微信支付商户号

| 项目 | 说明 |
|------|------|
| **申请条件** | 个体工商户营业执照即可 |
| **在哪申请** | https://pay.weixin.qq.com |
| **需要** | 营业执照 + 法人身份证 + 银行卡（对私账户就行） |
| **费率** | 0.6% |
| **审核时间** | 1-5 个工作日 |

### 4. 支付宝商户号

| 项目 | 说明 |
|------|------|
| **申请条件** | 个体工商户营业执照 |
| **在哪申请** | https://open.alipay.com |
| **需要** | 营业执照 + 法人身份证 + 银行卡 |
| **费率** | 0.6% |

## 六、完整清单（按顺序）

| 步骤 | 事项 | 耗时 | 费用 |
|------|------|------|------|
| ① | 办个体工商户营业执照 | 1-5 天 | 免费 |
| ② | ICP 备案（阿里云后台） | 1-20 天 | 免费 |
| ③ | 申请电子软著 | 1-2 周 | 免费 |
| ④ | 申请微信支付商户号 | 1-5 天 | 免费 |
| ⑤ | 申请支付宝商户号 | 1-5 天 | 免费 |
| ⑥ | App 上架 | 审核 1-7 天 | — |

**全程费用：0 元**（除了可能的地址挂靠费，几十块/年）

**最短时间：2-3 周**（①②可以同时进行）

## 七、注意事项

1. **经营范围一定要写"互联网信息服务"**，否则办不了 ICP 备案
2. **先办营业执照 → 再办 ICP 备案 → 再上架 App**，顺序不能反
3. **你的域名 linyang.asia** 要先完成实名认证（阿里云后台）
4. **服务器在国内**（你已经有阿里云美国弗吉尼亚的），但 App 备案需要**国内服务器**。如果你的 App 面向国内用户，建议再买一台国内轻量服务器（¥50/月左右）
5. **个体户也要报税**，不过很简单，季度销售额 30 万以下免增值税
6. **电子营业执照**和纸质的效力一样，推荐用电子版

## 八、常见问题

**Q: 一个人能办吗？**
A: 能，个体工商户就是一个人经营。

**Q: 没有实体地址怎么办？**
A: 很多城市支持"仅网络经营"，不需要实体店面。具体问当地市场监管局。

**Q: 在外地能办吗？**
A: 营业执照在经营所在地办理。如果你人在深圳、服务器在美国，一般在你户籍地或居住地办理。

**Q: 办完以后每年要做什么？**
A: 每年 1-6 月做一次"年报"（网上填一下就行），不年报会被列入异常。

**Q: 以后不想做了怎么办？**
A: 注销很简单，网上就能办。或者不管它，连续不年报会被自动注销（但建议正规注销）。

---

*建议先打电话 12315 或当地市场监管局，确认你所在地的具体要求和是否支持网上办理。*
