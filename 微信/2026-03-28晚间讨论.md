# 2026-03-28 晚间讨论

## 共创区设计讨论

### 官方话题机制
- 民间话题（任何人发起）→ 6天投票 → 锁定
- 官方话题（官方发起）→ 限定高分/验证用户参与 → 提供建议
- 官方看完整数据 → 决定最终方案 → 发布结果
- 完整闭环：用户发现需求 → 官方锁定 → 专家建议 → 官方决策 → 研发 → 上线 → 反馈

### 推荐库改进
- 话题分级（爆款/精华/普通/沉底），不每个都提取
- 评论筛选：只取高赞Top 10，控制AI成本
- 去重合并：相似话题关键词聚合
- 库分层：热库/温库/冷库，按热度分
- 自动清理：6个月无访问降级

### 专家验证系统
- 职业档案核实 → 获得对应领域"专家"标识
- 验证方式：证书上传/职业邮箱/社交验证/行业邀请码/共创贡献度
- 限定话题：仅验证用户可参与
- 专家评论权重更高，推荐库优先提取

### 商业变现路径
- 品牌方付费：话题冠名/官方推荐位/数据报告/首发合作
- 用户付费：专家问答/定制方案
- 人才变现：人才推荐/项目对接/专家顾问/猎头服务
- 核心收入来自品牌方，不是用户

### 人才库思路
- 验证用户 → 参与官方话题 → 积累声望 → 成长为专家
- 平台帮用户建立职业品牌（不是免费打工，是投资自己）
- 三层资产叠加：内容层 × 人才层 × 数据层

### 企业注册讨论
- 已有人才库和启动资金
- 是否直接注册企业（而非个体工商户）
- 建议：有资金+有合作伙伴→直接注册有限责任公司
